## Development

The `Makefile` contains all sorts of useful tasks.

### Running tests

```shell
make run-tests
```

### Creating code coverage report

```shell
make coverage-report
make coverage-report-server
```

### Building documentation

```shell
make build-docs
make docs-server
```
